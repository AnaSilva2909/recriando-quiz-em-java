- [ ] #####  AO FINAL, EXIBIR A PONTUAÇÃO TOTAL .

- [ ] ##### BOM SE FIZER ENTRE 50 E 74 PONTOS. 

- [ ] ##### REGULAR SE FIZER ENTRE 25 E 49 PONTOS. 

- [ ] ##### PÉSSIMO SE FIZER MENOS DE 24 PONTOS. 

- [ ] ##### APÓS, PODEM SER EXIBIDOS ERROS E ACERTOS.
